# 上海大学选课课表优化插件

## 功能简介

本插件用于上海大学教务系统课表页面，自动获取课程信息，并根据课程的上课周次为课表进行智能着色。支持以下功能：

- 自动识别并提取当前学期所有课程信息。
- 根据课程的上课时间和周次，智能为课表单元格着色。
- 支持同一时间段不同周次有不同课程的情况，自动合并显示为“全学期有课”。
- 课表上仅显示该时间段的课程名称，界面简洁明了。
- 仅在点击浏览器扩展按钮后弹出操作界面，避免页面自动弹窗干扰。

## 使用方法

1. 安装插件后，登录上海大学教务系统，进入课表页面。
2. 点击浏览器右上角的插件图标，弹出操作界面。
3. 插件会自动获取并着色当前学期的课表。

# Shanghai-University-Course-Selection-Schedule-Plugin

## Features

This plugin is designed for the Shanghai University academic system timetable page. It automatically retrieves course information and intelligently colors the timetable based on the weeks each course is held. Features include:

- Automatically identifies and extracts all course information for the current semester.
- Intelligently colors timetable cells based on course times and weeks.
- Supports cases where different courses are scheduled in the same time slot for different weeks, and automatically merges them as "Full Semester Course".
- Only displays course names for each time slot, keeping the interface clean and simple.
- The operation interface only pops up after clicking the browser extension button, avoiding automatic pop-ups on the page.

## Usage

1. After installing the plugin, log in to the Shanghai University academic system and enter the timetable page.
2. Click the plugin icon in the upper right corner of the browser to open the operation interface.
3. The plugin will automatically retrieve and color the timetable for the current semester.
